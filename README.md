# 자리톡 소통 커뮤니티 API
이 API는 임차인, 임대인, 공인중계사 및 외부이용자가
자유롭게 이용 할 수 있는 커뮤니티의 API입니다.

***

사용가능한 API 목록<br />
(외부 이용자는 글 등록을 하실 수 없습니다.)

    글 목록 호출
    글 등록
    글 수정
    글 삭제
    글 좋아요 누르기
    글 좋아요 해제하기
    댓글 쓰기
    댓글 수정
    댓글 삭제
